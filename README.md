# DBZ Legend of the Super Saiyan  Editor

## What Is it?
A game editor tool for a SNES game DBZ - Legend of the Super Saiyan (aka Densetsu). This tool allows you to mod the game through a visual user interface without any reverse engineering experience. 

## Setup
Clone the [Reverse Engineering Toolkit](https://github.com/MakoInfused/Reverse-Engineering-Toolkit) into a directory, then clone this repository into the same parent directory.
